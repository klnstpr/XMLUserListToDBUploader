Correct xml file syntax:

<users>

<user>

<name>name1</name>

<surname>surname1</surname>

<login>login1</login>

</user>

...

<user>

<name>nameN</name>

<surname>surnameN</surname>

<login>loginN</login>

</user>

</users>

Name, surname and login can be random. It is not required to specify it in the format: name1, surname1, login1 etc.
